# LuxyBerry

LuxyBerry é um MVP de plataforma digital para um negócio artesanal de caixas de morangos cobertos com chocolate, desenvolvido para fins de validação e aprovação.

## Visão geral
O projeto permite que clientes montem pedidos personalizados e realizem o pagamento online, enquanto a produtora recebe os pedidos já pagos para produção sob demanda.

## Tecnologias
- HTML, CSS, JavaScript
- Node.js
- Express
- Stripe (modo teste)

## Status
Projeto em fase de aprovação (MVP).
